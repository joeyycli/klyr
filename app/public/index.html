Three steps to anything.

No setup guides. No config files. Just a terminal and an idea.

01

Run klyr

Open a terminal and type klyr. It starts a local server, creates a workspace, and hands you a session ID.

02

Describe

Tell Claude what to build in plain English. A portfolio, a dashboard, a full SaaS app — whatever you need.

03

Ship

Watch files appear in real-time. Browse the code, preview the result live, iterate until it's right.

Every file stays on your machine. Every session is yours. The tools are local, the code is free, and nothing is locked behind a paywall.

klyr exists because building software should feel like a conversation — not a checkout page.

UNDER THE HOOD

Built on the best.

Claude Code

The same CLI developers use daily. klyr pre-configures it as a no-code builder so it scaffolds, installs, and writes production-grade code from the start.

Live Workspace

A browser-based terminal, file explorer, and live preview — all connected over WebSocket. Every keystroke streams in real-time.

Smart Defaults

Each session starts with a system prompt combining patterns from leading AI builders — React, TypeScript, Tailwind, and shadcn/ui out of the box.

Zero Cost

No subscription. No credits. klyr runs on your machine with your own Claude setup. The code is open source and always will be.
